# Budget Tracker Web Application — Project Brief

## Overview

A personal budgeting tool designed to track income and expenses, categorize transactions, set monthly spending limits, and monitor savings progress. The application provides clear financial insights through dashboards and visual charts.

## Core Features

### 1. Transactions Management

- Add income or expense entries with: amount, date, category, and description.
- Support for predefined and custom categories.
- Transaction history listing with filtering (category, date range, type) and sorting (date, amount).

### 2. Category Limits & Savings Goals

- Set monthly spending limits per category.
- Track current spending against limits.
- Define a monthly savings goal calculated as **total income - total expenses**.
- Display real-time progress toward savings goals.

### 3. Dashboard & Statistics

- Summary metrics: total income, total expenses, net balance.
- Time-based summaries (daily/weekly/monthly).
- Category distribution charts.
- Remaining budget estimation based on limits and spending.

## Additional Capabilities (Optional)

- Authentication (local or third‑party).
- Recurring income or expense entries.
- Smart insights for overspending and limit alerts.
- Calendar view of transactions.
- Import/export in CSV or JSON.
- Enhanced filters and full‑text search.
- Multi-currency support.

## UI/UX Notes

- Light/dark mode.
- Customizable category colors or icons.
